# Weight Tracker 25

A small project attempt to be generated, configured and deployed  using mostly AI.

The application should be a simple Weight Tracker, which can be self hosted on a system with relatively low resource requirements.

## Development environment

An average performance x86 computer for 2022.
8 GB or RAM.
Linux, Fedora based system.

## Target production environment

Intel N100 class computer (x86, 64bit)
4 GB or RAM.
Linux, Fedora based system.

## General software requirements

Use technologies that the developer is familiar with or eager to learn.
Currently favoring the following technology stack:

- .NET for back-end (version 10 or newer)
- Angular for front-end (version 20 or newer)
- sqlite for the DB
